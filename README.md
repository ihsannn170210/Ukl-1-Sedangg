# Program Faktorial java
Sebuah Program untuk mengihitung faktorial dari sebuah Bilangan bulat yang dimasukkan oleh pengguna
- Program meminta pengguna memasukkan bilangan bulat positif.
- Jika bilangan yang dimasukkan kurang dari 0, program akan memberi pesan bahwa faktorial tidak bisa dihitung.
- Jika bilangan valid, program akan menghitung faktorial dengan cara mengalikan angka dari 1 sampai bilangan tersebut.
- Hasilnya ditampilkan ke layar.
